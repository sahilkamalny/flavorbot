### Login System

FlavorBot includes a complete authentication system with secure login:

2. **Password hashing** before database storage
3. **Session management** via `SessionManager` singleton

## ğŸ¤– AI-Powered Recipe Generation

### The Generation Process

1. **Collect Ingredients**: Gather all items from ingredient list
2. **Load Preferences**: Fetch user's dietary restrictions and preferences
3. **Build Prompt**: Construct detailed prompt for GPT-3.5
4. **API Call**: Send request to OpenAI
5. **Display Result**: Show formatted recipe in text area

### Recipe Structure

Generated recipes include professional sections:

| Section | Content |
|---------|---------|
| **Ingredients** | Complete list with quantities |
| **Preparation** | Pre-cooking steps |
| **Cooking Steps** | Numbered instructions |
| **Tips** | Chef's tips for best results |
| **Serving Suggestions** | Plating and pairing ideas |

### Prompt Engineering

The AI receives context as a "professional chef with years of experience" to ensure:
- Appropriate skill-level instructions
- Respect for dietary restrictions
- Creative use of available ingredients
- Restaurant-quality presentation
